# Поиск адресов МКД

Веб-приложение для поиска адресов многоквартирных домов из базы данных.

## Функции
- Поиск адресов по названию улицы, города или части адреса
- Автодополнение при вводе
- Подробная информация по каждому адресу
- Адаптивный дизайн для мобильных устройств

## Технологии
- Flask (Python)
- HTML/CSS/JavaScript
- CSV база данных

## Локальная установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Поместите файл `export-reestrmkd-50-20251201.csv` в корневую папку
4. Запустите: `python app.py`
5. Откройте http://localhost:10000

## Развертывание на Render
1. Загрузите проект в GitHub
2. Создайте новый Web Service на Render.com
3. Укажите репозиторий
4. Настройки автоматически определятся из `requirements.txt`
5. Нажмите Deploy
